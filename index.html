Diseño del sistema

El diseño general del sistema de recepción de señales satelitales para la obtención de imágenes meteorológicas basado en dispositivo de radio definido por software (SDR), se presenta en el diagrama de bloques de la figura.

Sistema de Recepción

El bloque de recepción está formado por una antena de dipolo de media onda dimensionada de acuerdo a los requerimientos de la aplicación. Se hizo uso de este tipo de antena debido a su fácil fabricación y economía. Para la demodulación, amplificación y filtrado de la señal recibida en la antena, el dispositivo SDR que se utilizó es un Nooelec NESDR Mini SDR & DVB-T en conjunto con CubicSDR, un software para el uso y configuración de dispositivos SDR. Se eligió este software debido a que es de uso gratuito, intuitivo y fácil de ejecutar.

Procesamiento y Almacenamiento de Imágenes

La siguiente etapa corresponde al procesamiento de la señal recibida y su decodificación con la ayuda del software gratuito WXToImg, una herramienta muy útil que decodifica la señal de audio recibida por el Nooelec NESDR Mini SDR & DVB-T y entrega la imagen clara tomada por el satélite; imagen que puede almacenarse en una dirección cualquiera del PC indicada en la configuración del software. En esta etapa también se hizo uso de un cable virtual (VB-Audio Virtual Cable) que conecta la salida del software CubicSDR a WXToImg. De la misma manera que todos los softwares usados en este diseño, VB-Audio Virtual Cable es un software gratuito que puede instalarse libremente desde internet.

Envío a la Web

En esta última etapa se hizo uso del editor Atom para hacer el código fuente de nuestro sitio web y, gracias al servicio de repositorios, creación de páginas web y hosting gratuito de GitHub, se creó un repositorio con las imágenes tomadas y se diseñó la página web para el acceso remoto a la información captada.
